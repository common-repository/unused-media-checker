Identify and delete unused media files in your media library.

== Description ==
Unused Media Checker is a WordPress plugin that helps you identify and delete unused media files in your media library. It scans your media library and displays media files that are not attached to any published posts or pages, including post images. This plugin also considers media files used in drafts as unused and lists them accordingly, so you can easily remove them and optimise your WordPress website.

Unused Media Checker ist ein WordPress-Plugin, das Ihnen hilft, ungenutzte Mediendateien in Ihrer Mediathek zu identifizieren und zu löschen. Es scannt Ihre Mediathek und zeigt Mediendateien an, die nicht an veröffentlichte Beiträge oder Seiten angehängt sind, einschließlich Beitragsbilder. Dieses Plugin betrachtet auch Mediendateien, die in Entwürfen verwendet werden, als ungenutzt und listet sie entsprechend auf, so dass Sie sie leicht entfernen und Ihre WordPress-Website optimieren können.

== Installation ==
Download the plugin ZIP file from WordPress plugin directory.
Upload the plugin ZIP file to your WordPress site by navigating to Plugins > Add New > Upload Plugin and selecting the file.
Activate the plugin through the Plugins menu in WordPress.
	
Laden Sie die ZIP-Datei des Plugins aus dem WordPress-Plugin-Verzeichnis herunter.
Laden Sie die Plugin-ZIP-Datei auf Ihre WordPress-Website hoch, indem Sie zu Plugins > Add New > Upload Plugin navigieren und die Datei auswählen.
Aktivieren Sie das Plugin über das Plugins-Menü in WordPress.
